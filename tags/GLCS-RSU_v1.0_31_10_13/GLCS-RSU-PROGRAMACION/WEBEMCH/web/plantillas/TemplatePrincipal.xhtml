Bienvenido al Sistema de Getión de Liquidaciones y Control de Servicio para la Empresa Empresa Emch Servicios Ecologicos(GLCS-RSU). Aqui como Nuestro Usuario podra tomar acciones para la administracion de documentos, personal y camiones. Tambien tendra la posiblidad de llevar un control adecuado de los documentos generados. En nuestro sistema tambien se puede tener un control de incidencias de los camiones que estan trabajando. Ademas con nuestro sistema se controlará los estados de los documentos como liquidaciones y comprobante de pago.

Este es el módulo encargado de la seguridad del sistema en donde un administrador podra crear eliminar o editar a un usuario.Tambien un usuario normal podra cmabiar su propia contraseña.

En este módulo el usuario podra crear, eliminar, editar o consultar a trabajadores, camiones, empresas clientes o verificar comprobantes.

En este módulo el usuario podra crear hojas de despacho, hojas de control de viaje, asignar rutas a camiones, consultar las rutas asignadas, llevar un control de las asistencias e incidencias ocurridas.

En este módulo se lleva el control de comprobantes, liquidaciones y reportes parte en la que el sistema realiza cálculos para sacar resultados finales para la empresa.
